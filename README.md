# Contra

> `cat` but with line numbers, more like `bat` but not really :)

## Spec
> Running on `zig v0.16.0-dev.683+60a332406`

## Running
```sh
  # Build and run
  make build run
  # OR
  zig build run

  # Run build after building
  make run

  # Build C
  make build-c

  # Run C
  make run-c
```
